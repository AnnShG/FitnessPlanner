# Fitness Planner
## Fitness Planning Calendar with AI Integration

An Android fitness application that enables users to take control of their workout journey. This application offers flexible, 
calendar-based planning and personalized AI-driven insights. 
This project is part of a Bachelor’s Thesis in Computer Science.

## Key Features
- Interactive Calendar - A flexible planning system where users can plan, track, and review past, present, and future activities
- Customization - Creating custom exercises and workouts with a specific colour indicator
- Non-judgmental Support - Optional motivational quotes to help maintain consistency without the pressure of strict streak metrics
- AI-Driven Insights - Integration with Gemini AI to generate personalized training recommendations based on weekly activity patterns
- Progress Visualization - Interactive charts to track personal metrics and improvement
